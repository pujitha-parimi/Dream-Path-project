# ✈️ Dream-Path

## 📌 Overview

Dream-Path is a web-based platform designed to guide students who aspire to pursue higher education abroad. The application provides information about universities, courses, countries, admission processes, scholarships, and career opportunities in an easy and user-friendly way.

The goal of this project is to simplify the study abroad journey and help students make informed decisions about their future.

---

## 🚀 Features

✅ User-Friendly Interface  
✅ Study Abroad Guidance  
✅ University and Course Information  
✅ Country-wise Details  
✅ Scholarship Information  
✅ Responsive Design  
✅ Easy Navigation
